# Tier 3 집중 문제 해결: 시그모이드 기울기 강화 + 티어 경계값 조정

## Context

안전장치 제거 후에도 Tier 3 비율이 과도하게 높음.
근본 원인: 승률 시그모이드(slope=20)가 49~51% 구간에서 점수 차이를 10점밖에 만들지 못하고,
Tier 3 구간(45~55)이 10점으로 넓어 대부분의 챔피언이 이 안에 몰림.

## 변경 내용

### 1. 시그모이드 기울기 강화 (slope 20 → 40)

```
// Before
winRateScore = 100 / (1 + exp(-20 * (adjustedWinRate - 0.50)))

// After
winRateScore = 100 / (1 + exp(-40 * (adjustedWinRate - 0.50)))
```

| 승률 | slope=20 | slope=40 |
|------|----------|----------|
| 48%  | 40.1     | **31.0** |
| 49%  | 45.0     | **40.1** |
| 50%  | 50.0     | 50.0     |
| 51%  | 55.0     | **59.9** |
| 52%  | 59.9     | **69.0** |

49~51% 구간 점수 차이: 10점 → **20점**으로 2배 확대.

### 2. 티어 경계값 재배치 (Tier 3 구간 축소)

```
// Before (10점 균등)
OP >= 75, 1 >= 65, 2 >= 55, 3 >= 45, 4 >= 35, 5 < 35

// After (Tier 3 = 6점으로 축소)
OP >= 80, 1 >= 65, 2 >= 53, 3 >= 47, 4 >= 35, 5 < 35
```

| 티어 | Before 구간 | After 구간 |
|------|------------|-----------|
| OP   | 75~100 (25pt) | **80~100 (20pt)** |
| 1    | 65~75 (10pt)  | **65~80 (15pt)**  |
| 2    | 55~65 (10pt)  | **53~65 (12pt)**  |
| 3    | 45~55 (10pt)  | **47~53 (6pt)**   |
| 4    | 35~45 (10pt)  | **35~47 (12pt)**  |
| 5    | 0~35 (35pt)   | 0~35 (35pt)       |
